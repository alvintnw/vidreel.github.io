1. Daftar developer di developers.tiktok.com
2. Buat app baru → aktifkan Content Posting API
3. Tambahkan scope: video.upload dan video.publish
4. Lakukan OAuth flow dengan Client Key Anda untuk mendapatkan access token
5. Paste token di form di bawah ini
